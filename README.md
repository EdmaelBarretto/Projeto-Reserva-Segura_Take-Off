# Projeto Reserva Segura 💰

Aplicativo de **educação financeira gamificada** que incentiva usuários a pouparem dinheiro com disciplina.

---

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte da **Residência Tecnológica do Porto Digital - Programa Take Off**.

---

## 📌 Problema

Muitos brasileiros têm dificuldade em:
- Poupar dinheiro  
- Criar uma reserva de emergência  
- Manter disciplina financeira  

---

## 💡 Solução

O **Reserva Segura** transforma o ato de poupar em um jogo:

- 💰 Depósitos reais  
- 🎯 Missões e desafios  
- ⭐ XP e badges  
- 🏆 Ligas e ranking  
- 🎁 Recompensas reais  

---

## 🎯 Público-Alvo

- Pessoas sem reserva de emergência  
- Interessados em educação financeira  
- Usuários conectados à tecnologia  

---

## ⚙️ Funcionalidades Iniciais

- Criar meta de poupança  
- Realizar depósitos  
- Acompanhar evolução  
- Sistema básico de gamificação (XP e níveis)  

---

## 🧱 Tecnologias (proposta)

- Back-end: Java + Spring Boot  
- Front-end: React  
- Banco de dados: PostgreSQL  

---

## 🎯 Objetivo

Ajudar pessoas a desenvolverem o hábito de poupar e conquistarem segurança financeira.
